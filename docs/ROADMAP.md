# プロジェクトロードマップ: オリウマレース集計ツール

## 概要
本ロードマップは、オリウマレース集計ツールの開発フェーズとマイルストーンを定義する。
開発は「Core Logic First」および「MVP Prototype Verification」のアプローチを採用し、早期に実戦投入可能な状態を作ることを優先する。
